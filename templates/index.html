Cloud Computing Catalog Service

Author: Tejit Pabari

Api Paths

Catalog: /catalog

Description: Shows the catalog of top 5 tutors for selected categories

Tutors: /tutors

Description: Shows list of tutors.
Can limit and offset it. /tutors?limit=5&offset=2
Can take various fields as well /tutors?name=Gitika%20Bose

Tutors id: /tutors/1

Description: Shows a tutor by id. Can take in various fields as well /tutors/1?fields=name,categories,byline,experience

Categories: /categories

Description: Shows list of categories.
Can limit and offset it. /categories?limit=2&offset=1

Categories id: /categories/1
